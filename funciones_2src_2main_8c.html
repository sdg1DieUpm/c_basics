Detailed Description

Main file on functions in C.

Code about the use of functions and passing of argument by reference in C.

The code is a brief explanation on the basic use of functions and how to pass arguments by reference using memory addresses and pointers. This is a very common example on how a code of the subject SDG2 (the lab of Sistemas Digitales) looks like. This code is focused to the introduction of C to the students of Sistemas Digitales of the ETSI Telecomunicacion UPM. For the sake of a better comprehension of the students of the current degree, the inline comments and the explanations printed out by the printf function are written in Spanish.
