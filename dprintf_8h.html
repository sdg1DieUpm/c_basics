Detailed Description

Enable or disable debug comments.

By defining the macro DEBUG we enable to print those lines intended to be debug comments that use the macro DPRINTF(). If the macro DEBUG is not defined, those lines intended to be debug comments using the macro DPRINTF() will not be printed on the standard output.
