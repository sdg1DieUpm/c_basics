Detailed Description

Main file on precompile directives in C.

Code about the precompile directives in C #include, #define, #if, #ifndef, #undef, #error.

The code is a brief explanation of each one of the directives mentioned above. Oriented to the introduction of C to the students of Sistemas Digitales of the ETSI Telecomunicacion UPM. It is strongly recommended to comment and uncomment the #includes of the different parts of the explanations to be able to experiment with the code in order to make the terminal output more friendly.

For the sake of a better comprehension of the students of the current degree, the inline comments and the explanations printed out by the printf function are written in Spanish.
